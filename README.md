# tamahackathon
Wellness apps retain users with guilt; Gotchu retains them with forgiveness. It is a desktop AI pet that can't die, remembers only with consent, and evolves through trust rather than attendance. Under the hood: live inference with a three-tier fallback, consent-gated semantic memory, governed pet-to-pet friend nudges, and a crisis detector that ships with its own test suite.
